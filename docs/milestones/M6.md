# M6 - Datenaggregation

## Status

Completed on 2026-04-24.

## Implemented

- Added raw-odds aggregation service.
- Groups odds by player, game, market, and line.
- Pairs Over/Under and Yes/No sides per bookmaker.
- Removes vig when both sides are available.
- Calculates consensus probabilities and expected values.
- Writes results to `aggregated_odds`.
- Added `POST /aggregations/refresh`.

## Notes

Name normalization currently uses the source-level `external_id` strategy introduced in M4. Cross-book fuzzy matching will become more important once multiple live books provide overlapping player props.
