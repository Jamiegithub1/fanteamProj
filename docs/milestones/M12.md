## Current Source Reality

The application is stable, but live projection output depends on bookmaker availability:

- Playzilla discovery works with lightweight HTTP, but current player-prop payload extraction is degraded because the available lightweight payload contains no NBA player props.
- DraftKings adapter is implemented and disabled by default because the public sportsbook edge returns HTTP 403 from this server.

Both cases are stored in `source_health` and do not crash the app.
