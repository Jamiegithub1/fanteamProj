# M3 - Odds Math Engine

Datum: 2026-04-24

## Ergebnis

- American Odds werden in implizite Wahrscheinlichkeiten umgerechnet.
- Decimal Odds werden in implizite Wahrscheinlichkeiten umgerechnet.
- Over/Under-Wahrscheinlichkeiten werden vig-frei normalisiert.
- Expected Values werden reproduzierbar aus Line und Over/Under-Wahrscheinlichkeit approximiert.
- Aggregation berechnet Mean und Median.
- Outlier Detection nutzt Median Absolute Deviation.
- Confidence Score kombiniert Quellenanzahl, Outlier und Streuung.

## Entscheidungen

- Die Engine ist reine Python-Logik ohne Datenbankzugriff, damit sie isoliert testbar bleibt.
- `Decimal` wird fuer Quoten, Wahrscheinlichkeiten und erwartete Werte verwendet.
- Die Expected-Value-Approximation verschiebt die Line um maximal einen konfigurierbaren Half-Point-Step pro voller Over/Under-Schieflage. Das ist bewusst konservativ und ohne historische Daten.
- Outlier werden vor der Aggregation entfernt, bleiben aber im Ergebnis sichtbar.
