# M7 - Projection Engine

## Status

Completed on 2026-04-24.

## Implemented

- Added FanTeam projection engine.
- Converts aggregated stat expected values into fantasy points.
- Handles Double-Double and Triple-Double as probability-weighted bonuses.
- Uses one projection row per player for the next available game slice.
- Writes results to `projections`.
- Added `POST /projections/refresh`.
